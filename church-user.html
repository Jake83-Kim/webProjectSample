USER FEATURES

포함 기능

요청하신 신도용 앱 기능을 실제 사용자 여정 중심으로 다시 정리했습니다.

사용자 계정 기능

회원가입, 로그인, 아이디/비밀번호 찾기, 소셜 로그인, 신원 인증 및 가입 프로세스를 포함합니다.

01
편리한 접근성

일반 로그인과 함께 소셜 로그인으로 빠르게 진입할 수 있습니다.

02
가입 인증 프로세스

교회 소속 확인과 신원 인증 절차를 거쳐 안전하게 가입합니다.

커뮤니티 기능

커뮤니티 전체 조회 및 검색, 카테고리별 게시글 보기, 커뮤니티 상세 보기, 좋아요, 스크랩 기능을 제공합니다.

01
탐색과 검색

카테고리, 인기순, 최신순 기준으로 게시글과 커뮤니티를 살펴볼 수 있습니다.

02
참여 액션

좋아요와 스크랩을 통해 관심 글을 다시 모아보고 개인화된 피드를 구성합니다.

일정 및 행사 보기 / 교회 소개

교회 일정과 행사 목록을 보고, 교회 소개 페이지에서 위치와 주요 안내를 확인할 수 있습니다.

01
일정 목록 보기

예배, 행사, 봉사 일정과 참석 정보를 달력형 또는 리스트형으로 제공합니다.

02
교회 소개 보기

교회 비전, 예배 시간, 위치, 연락처 등 기본 정보를 사용자 친화적으로 안내합니다.

사용자 마이페이지

스크랩 목록 보기, 사용자 프로필 수정 등 사용자가 자신의 활동과 개인 정보를 관리할 수 있습니다.

01
스크랩 모아보기

저장한 게시글과 공지, 행사 정보를 마이페이지에서 다시 확인합니다.

02
프로필 수정

닉네임, 프로필 이미지, 관심 그룹 등 기본 프로필을 직접 수정할 수 있습니다.

플랫폼 고객지원 CS 기능

FAQ, 플랫폼 공지사항, 1:1 문의 기능, 약관 열람 기능을 앱 안에서 자연스럽게 이어지도록 배치합니다.

01
빠른 해결

FAQ와 공지사항을 먼저 제안해 자주 발생하는 문의를 빠르게 해소합니다.

02
1:1 문의

개별 문의가 필요한 경우, 앱 내 문의 등록과 답변 확인 흐름을 제공합니다.

콘셉트 포인트

밝은 배경, 강한 CTA 버튼, 둥근 칩, 선명한 GNB 포인트를 사용해 친숙하고 가벼운 커뮤니티 앱 인상을 만들었습니다.

01
밝은 톤

흰색과 미색 바탕 위에 오렌지 포인트를 얹어 접근성을 높였습니다.

02
직관적 위계

버튼, 칩, 탭을 한 가지 언어처럼 반복해 사용자가 금방 익숙해지도록 구성했습니다.

USER FLOW

신도 가입 및 참여 흐름

처음 들어오는 사용자 기준으로 가입부터 커뮤니티 참여까지의 흐름을 3단계로 요약했습니다.

01

회원가입 및 로그인

회원가입, 로그인, 아이디/비밀번호 찾기와 소셜 로그인을 통해 빠르게 앱에 진입합니다.

02

신원 인증 및 가입 완료

소속 교회와 기본 정보를 확인하는 인증 절차를 거친 뒤 정식 사용자로 가입을 마칩니다.

03

커뮤니티 탐색과 활동

게시글을 읽고 좋아요/스크랩을 남기며, 일정과 행사 정보를 확인하고 마이페이지를 관리합니다.
