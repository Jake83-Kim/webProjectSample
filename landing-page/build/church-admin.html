ADMIN FEATURES

포함 기능

요청하신 관리자 기능을 운영 업무 단위로 묶어서 카드로 정리했습니다.

교회 계정 기능

회원가입, 로그인, 아이디 찾기, 비밀번호 찾기, 교회 소개 관리까지 교회 단위 계정 운영을 지원합니다.

01
기본 계정 흐름

가입, 로그인, 아이디/비밀번호 찾기를 관리자 웹 기준으로 제공합니다.

02
교회 소개 관리

교회 소개 문구, 위치, 연락처, 예배 정보 등 소개 페이지를 직접 관리합니다.

회원 관리 기능

계정 전체 조회 및 검색, 회원 상태 변경, 신원 확인과 가입 승인 프로세스를 한 화면 흐름으로 구성합니다.

01
회원 검색/조회

가입 상태, 그룹 여부, 활동 이력 기준으로 빠르게 검색하고 필터링합니다.

02
승인/보류/반려

신원 확인 결과에 따라 가입 승인 프로세스를 관리자가 즉시 처리할 수 있습니다.

교회 그룹 관리 기능

그룹 조회 및 검색, 그룹 생성, 그룹 리더/구성원 설정 등 공동체 운영에 필요한 편집 기능을 포함합니다.

01
그룹 생성

청년부, 셀 모임, 봉사팀 등 다양한 교회 그룹을 만들고 설명을 설정합니다.

02
회원 설정

구성원 배정, 리더 지정, 참여 상태 변경까지 그룹 내부 권한을 조정합니다.

커뮤니티 관리 기능

커뮤니티 목록과 상세 확인, 문제 상황 대응을 위한 신고/차단 시스템, 카테고리 관리까지 모두 지원합니다.

01
게시글 운영

커뮤니티 목록 보기, 상세 보기, 카테고리별 현황 확인이 가능합니다.

02
안전 대응

신고 접수, 게시글 차단, 사용자 차단 등 즉시 대응 가능한 운영 도구를 제공합니다.

공지 / 일정 / 행사 관리

공지 목록 보기, 공지 상세 보기, 공지 등록/수정과 일정 목록 보기, 일정 등록/수정을 하나의 운영 묶음으로 제공합니다.

01
공지 기능

교회 공지, 예배 안내, 행사 알림을 등록하고 수정할 수 있습니다.

02
일정 및 행사 기능

예배 일정, 행사 일정, 모집 마감일을 운영자가 직접 갱신합니다.

통계 조회 및 플랫폼 고객지원 CS

운영 효율을 위해 활동 지표를 확인하고, 플랫폼 차원의 고객지원 기능도 함께 확인할 수 있도록 구성했습니다.

01
통계 조회

회원 증가, 승인 대기, 게시글 추이, 그룹별 활동량 등을 대시보드 형태로 확인합니다.

02
고객지원 CS

FAQ, 플랫폼 공지사항, 1:1 문의 기능, 약관 열람 기능을 관리자 웹에서도 제공합니다.

APPROVAL FLOW

회원 승인 프로세스

관리자가 실제로 많이 쓰게 될 흐름을 3단계 카드로 단순화했습니다.

01

가입 신청 확인

신청자의 기본 정보, 소속, 인증 자료를 검토하고 승인 대기 목록에서 빠르게 정리합니다.

02

신원 확인 및 상태 변경

검증 결과에 따라 승인, 추가 확인 요청, 반려 상태를 즉시 반영합니다.

03

그룹 및 커뮤니티 연결

승인 완료 후 기본 그룹 배정, 권한 설정, 커뮤니티 접근 범위를 연결해 운영 부담을 줄입니다.
